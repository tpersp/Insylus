Token Permissions

Recommended Proxmox Role

Create the token in Proxmox under Datacenter -> Permissions -> API Tokens.

For inventory and status, assign PVEAuditor on the target cluster or node path.

For start, stop, and restart, the token also needs VM.PowerMgmt on the VM/LXC path. PVEAdmin works, but it is broad. Prefer a custom role when practical.

Keep privilege separation enabled unless you intentionally want the token to inherit the user's full permissions.
